Bas. Client side integration testing made less painful.

Behaviour Assertion Sheets are a new way to assert to a test library how the front-end aspects of your website are intended to fit together. The syntax is familiar to anybody who has ever used CSS - simple, declarative, no cruft.

Bas (pronounced ‘bass’) is written in JavaScript, and runs in the server atop Node.js. Iâ€™m hoping that other people contribute new, better implementations for different stacks - and help specify Bas as a defacto standard.
